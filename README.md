A simple REST API written in Go. It is currently under development.

***

## Dependencies
- [Go](https://go.dev/)
- [Docker](https://www.docker.com/)

***

## Getting Started
1. Clone the repository
```bash
git clone https://github.xom/ellioht/go-rest-api.git
```

2. Build the docker container
```bash
make go-rest-api
```
